# Minecraft Discord Bot

Discord бот для обработки заявок на добавление в белый список для Minecraft серверов

## Текущий статус

Версия: v0.1.1

## Планы (MVP)

- Slash команда `/apply`
- Форма заявки
- Канал для персонала
- Кнопки "Принять/Отклонить"
- SQLite
- Изменение роли при утверждении заявки
